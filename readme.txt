Модуль предназначен для более информативного представления банеров в виде слайдшоу с активними маркерами.

Для установки модуля необходимо:
- скопировать файлы с архива в корень сайта;
- дать права на редактирование моодуля Система -> Пользователи -> Группы Пользователей;
- создать банер Дизайн -> Точечнный баннер;
- активировать модуль Дополнения -> Модули -> Слайдшоу c маркерами;
