The module is designed for a more informative presentation of banners in the form of a slideshow with active markers.

To install the module you need:
- copy files from the archive to the root directory of the site;
- give permissions to edit module: System -> Users -> User Groups;
- create banner: Design -> Dot banner;
- activate the module: Add-ons -> Modules -> Slideshow with markers;
